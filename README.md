# 🛒 ShopEZ – E-Commerce Frontend Application

ShopEZ is a fully functional **frontend e-commerce web application** built using modern web technologies.  
It demonstrates how a real-world shopping interface works — including product browsing, cart management, and checkout flow — all implemented on the client side.

This project is designed as a learning project to understand how frontend applications handle data, UI, and user interactions before integrating with a backend.

---

## 📌 Project Overview

ShopEZ allows users to:

- Browse a list of products  
- View detailed product information  
- Add products to a shopping cart  
- Remove items from the cart  
- View total cart value  
- Simulate a checkout process  

The entire application runs **only in the browser**, without any backend server or database.

---

## 🎯 Objectives

The main goal of this project is to:

- Build a real-world style e-commerce UI  
- Understand dynamic content rendering using JavaScript  
- Learn how to manage application state using LocalStorage  
- Practice modular frontend development  
- Prepare frontend structure for future backend integration

## 🚀 Features

### 🛍️ Product Browsing
- Displays products dynamically from a JSON file  
- Shows product image, name, and price  
- Clean card-based layout using Bootstrap  

---

### 🔎 Product Details Page
- Displays full product information  
- Shows product image, description, and price  
- Includes **key features list** for better UX  
- Add to cart functionality  

---

### 🛒 Shopping Cart
- Add products to cart  
- Remove products from cart  
- Update product quantity  
- Automatically calculate total price  
- Cart data stored in LocalStorage  

---

### 💳 Checkout System
- Form to collect user details (Name, Email, Address)  
- Displays order summary  
- Simulates order placement  
- Clears cart after checkout  

---

### 🎨 UI / UX Features
- Responsive design using Bootstrap grid  
- Smooth hover effects on product cards  
- Loading spinner while fetching products  
- Toast notifications for user actions  
- Empty cart UI for better experience

## ⚙️ How to Run the Project

1. Download or clone this repository  
2. Open the project folder in **Visual Studio Code**  
3. Install the **Live Server extension**  
4. Right-click on `index.html`  
5. Click **"Open with Live Server"**
